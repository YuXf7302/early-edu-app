# 宝宝早教计划

3 岁男宝早教网页引擎。移动端优先，大字模式，适合老人使用。

## 数据说明（重要）

数据存储在**每个人的手机/电脑本地**，互不干扰：
- 妈妈在手机上设置孩子档案 -> 只有妈妈手机能看到
- 奶奶在平板上打卡 -> 只有奶奶平板有记录
- 爸爸在电脑上看周计划 -> 爸爸的数据完全独立

**不需要注册账号，也没有服务器数据库。** 每人打开链接后自己设置、自己使用。

## 部署方式（选一种即可）

### 方案 A：部署到 Vercel（推荐，免费，5 分钟）

1. 打开 https://vercel.com
2. 点 "Sign Up" -> 用 GitHub / Google / 邮箱注册
3. 登录后点 **"Add New – Project"**
4. 点 **"Deploy without Git"**
5. 把本项目文件夹里的 `deploy-early-edu.zip` **解压**，把里面所有文件**拖进去**
6. 点 "Deploy" -> 一两分钟后就有公网链接了
7. 把链接发给家人，家人点开就能用

### 方案 B：本地启动（仅限同一 WiFi）

1. 双击 `start-server.bat`
2. 电脑打开 http://localhost:5173
3. 手机连同一 WiFi，打开 http://172.16.1.6:5173

## 功能

| 页面 | 功能 |
|------|------|
| 今日 | 查看今天的活动，打卡完成 |
| 计划 | 生成周计划，每天活动可换可重新生成 |
| 资源 | 绘本推荐 + 老人活动指引 |
| 看板 | 每周复盘，完成率统计 |
| 设置 | 孩子档案 + 字体大小 |

## 技术栈

React 19 + Vite 8 + TypeScript 6 + Zustand 5 + Dexie(IndexedDB) + Tailwind CSS 3
